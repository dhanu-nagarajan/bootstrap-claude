# Showcase: Real Generated Output

This directory contains actual `.claude/` systems generated by bootstrap-claude on real project types. Use these to evaluate output quality before installing.

## Available Showcases

### Next.js Application (`nextjs-app/`)

A typical Next.js 14 application with App Router, Prisma ORM, and NextAuth.
Shows what bootstrap-claude generates for a TypeScript web application.

**Tier:** standard (12 files)

### FastAPI Service (`fastapi-service/`)

A FastAPI REST API with SQLAlchemy, Pydantic, and pytest.
Shows what bootstrap-claude generates for a Python backend service.

**Tier:** standard (12 files)

### Rust CLI (`rust-cli/`)

A Rust CLI tool built with clap, serde, and tokio.
Shows what bootstrap-claude generates for a Rust command-line application.

**Tier:** lite (4 files)

## How to Read These

Each showcase contains:
- `README.md` — Annotated walkthrough explaining the generated files
- `.claude/` — The actual generated output (what you'd get from running `/bootstrap`)

## Quality Bar

Every file in these showcases meets the quality bar:
> Would a principal engineer find this useful, or delete it as noise?

Every reference points to a plausible file in the described project. Every rule is specific and actionable. No generic advice.
